# Управление несколькими выходами с разными частотами на Arduino

## Описание решения

Программа управляет пятью выводами микроконтроллера (PORTB, пины 1-5), генерируя на каждом сигнал с индивидуальной частотой переключения. Используется 16-битный таймер Timer1 в режиме CTC (Clear Timer on Compare Match) для создания точных временных интервалов.

## Принцип работы

Каждый пин имеет свой счётчик, который инкрементируется при каждом прерывании таймера. Когда счётчик достигает заданного порога, состояние пина инвертируется, а счётчик сбрасывается. Пороговые значения для разных пинов различаются, что создаёт разные частоты переключения.
